# TihmstarInstaller

## What does it do?
* It automates the bulding of tihmstar packages and installs them!

## What does it build as of right now?

* libgerneral 
* img4tool
* liboffsetfinder64
* xpwn
* libipatcher
* libfragmentzip
* libirecovery
* libplist
* ra1nsn0w
* libimobiledevice
* futurerestore
* idevicerestore
