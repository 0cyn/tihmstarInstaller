# TihmstarInstaller

## What does it do?
* It automates the bulding of tihmstar packages and installs them!

## Usage notes

* Usage: tihmstarInstaller <arg>
* -c              Just download and compile all the software
* -d              Download all depends
* -a              Do all
* -u              Updates all packages
* -h              Shows this help
